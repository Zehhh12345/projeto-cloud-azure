# Projeto Cloud Azure

## Descrição
Projeto prático utilizando Microsoft Azure com criação de duas máquinas virtuais (Windows e Linux), configuração de rede, acesso remoto e testes de segurança.

## Arquitetura
- VM Windows Server (10.0.0.4)
- VM Ubuntu (10.0.0.5)
- Rede: VNet Azure

## Funcionalidades
- Acesso via RDP e SSH
- Comunicação entre VMs
- Servidor web (IIS e Python)
- Configuração de firewall (NSG)
- Testes com Nmap

## Segurança
Testes de portas abertas e controle de acesso via NSG.
